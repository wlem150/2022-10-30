단어 관련 속성

요소 안에 공백을 어떻게 처리할지 지정하는 속성이다. white-space: normal | nowrap | pre | pre-line | pre-wrap | initial | inherit;
  • normal : 공백과 개행을 무시하고, 필요한 경우에 자동 줄바꿈 발생. 기본값
  • nowrap : 공백과 개행을 무시하고, 자동 줄바꾸이 일어나지 않음
  • pre : 공백과 개행을 표현하고, 자동 줄바꿈이 일어나지 않음.
  • pre-line : 공백은 무시하고, 개행만 표현, 필요한 경우에 자동 줄바꿈 발생.
  • pre-wrap : 개행은 무시하고, 공백만 표현 필요한 경우 자동 줄바꿈 발생
letter-spacing: normal | length | initial | inherit;
  • normal : 기본값
  • length(px) : 픽셀로 글자간 간격 설정, 음수사용시 겹쳐짐
word-spacing: normal|length|initial|inherit;
  • normal : 기본값
  • length : 뛰어쓰기 공백을 얼마나 할 것인지 픽셀단위로 설정함. 음수사용시 단어간 겹쳐짐
word-break: normal | break-all | keep-all | initial | inherit;
  • normal : 기본값, 중단점은 공백이나 작대기(-)(CJK는 음절)
  • break-all : 중단점은 음절. 모든 글자가 요소를 벗어나지 않고 개행
  • keep-all : 중단점은 공백이나 작대기(-)
word-wrap: normal|break-word|initial|inherit;
  • normal : 기본값, 중단점에서 개행
  • break-word : 모든 글자가 요소를 벗어나지 않고 강제로 개행

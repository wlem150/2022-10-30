Display

Display : value;
  • none : 아무것도 안보이게 만듬
  • inline : 박스를 인라인으로 만듬. 위에 부모요소를 벗어남
  • block : 부모요소 밖으로 튀어나가지 않고 안으로 박스가 만들어지며, 블랙형태로 만들어짐
  • inline-block : inline과는 다르게 부모요소 안에서 박스가 인라인형태로 만들어짐 * height 나 width 등과 같은 박스모델 속성을 적용할 수 있다.
  • list-item : block이랑 같지만 왼쪽에 ul 리스트와 같은 점이 각각의 라인마다 찍여져 나온다.
  • 그외에 list-item, flex, inline-flex, table, table-cell 등 다양한 속성 값 존재
  • inline level 요소 사이의 공백과 개행 처리 inline 요소의 경우 공백과 개행에 대해서 하나의 여백으로 받아들인다. 따라서 inline 과 inline-blcok 의 경우 태그 사이의 공백이나 개행이 있을 경우 약 4px의 여백을 가지게 된다.

text-decoration

text-decoration: text-decoration-line text-decoration-color text-decoration-style | initial | inherit;
  • none : 택스트 꾸밈을 생성하지 않음(기본값)
  • underline : 밑줄로 꾸밈
  • overline : 윗줄로 꾸밈
  • line-through : 중간을 꾸밈
  • solid : 한줄 스타일(기본값)
  • double : 이중선 스타일
  • dotted : 점선 스타일
  • dashed : 파선 스타일
  • wavy : 물결 스타일

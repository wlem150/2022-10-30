Clear

Clear 속성은 float 속성을 사용한 후 의도치 않은 결과가 나타났을 때, 보통 사용된다.

clear: none | left | right | both | initial | inherit;
  • none : 양쪽으로 float 요소를 허용(기본값)
  • left : 왼쪽으로 floating 요소 허용하지 않음
  • right : 오른쪽으로 floating 요소 허용하지 않음
  • both : 양쪽으로 floating 요소 허용하지 않음

가상클래스

가상 클래스는 미리 정의해놓은 상황에 적용되도록 약속된 보이지 않는 클래스이다. 직접 요소에 클래스를 선언하는 것이 아니라, 약속된 상황이 되면 브라우저가 알아서 클래스를 적용해 준다.

예를 들어, p태그에 마우스 커서를 올려놓았을 때 특정 스타일을 주고 싶다고 했을 시에 가상클래스를 사용한다.

가상클래스
  1. 임의의 클래스 선택자를 선언하여 특정 스타일 규칙을 만든다.
  2. p태그에 커서가 올라가면 p태그에 클래스를 집어넣는다.
  3. p태그에서 커서가 빠지면 p태그에 클래스를 제거한다.

가상 클래스는 ":"을 사용해서 나타낼 수 있다.
